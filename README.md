# 책보리 🌱
'책보리'는 서울시의 공공헌책방 ‘서울책보고’에서 진행되는 북크로싱에 참여할 수 있는 서비스입니다. 충분히 가치 있는 헌책들이 순환되지 못하고 개개인의 책장에 방치되고, 폐기되는 상황이 안타까웠습니다. 앱 ‘책보리’를 통해 다른 사람이 인상 깊게 읽은 책을 자신의 도서 1권과 손쉽게 교환할 수 있습니다.
